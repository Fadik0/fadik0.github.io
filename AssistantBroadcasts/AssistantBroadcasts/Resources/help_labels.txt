-----Заметки-----

Имена файлов соответствуют именам заметок.
Файлы заметок сохраняются в папке
C:\Users\<Имя пользователя>\Documents\Assistant Broadcasts

Если список содержит две заметки с одинаковым именем, то в файл запишутся данные той которая ниже по списку.

Управлять позицией в списке можно с помощью стрелок в правом нижнем углу. Там же есть кнопка для переключения отображения между именем и описанием.
