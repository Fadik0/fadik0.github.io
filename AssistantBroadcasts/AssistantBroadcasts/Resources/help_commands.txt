-----Команды-----

Стандартные команды:
!help - выводит список команд, кроме тех которые отмечены скрытыми, доступна всем;
!bot_sleep - отправляет бота в сон и оповещает сообщением, которое можно изменить в настройках, доступна стримерам, помощникам и модератором.
!quest - команда нужная для работы заданий, не рекомендуется ее переназначать.

Стандартные команды можно переназначить для этого нужно создать новую команду со стандартным именем и, например, нужным текстом.

Если список содержит две команды с одинаковым именем, то выведется та которая выше по списку.

Управлять позицией в списке можно с помощью стрелок в правом нижнем углу. Там же есть кнопка для переключения отображения между именем и описанием.

Скрытая функция если не установлена галка regex и игнорирование регистра, то в поле ответ можно на писать тэг {parametr}, который будет заменен вторым словом из сообщения.
Например:
Команда - !snawball
Сообщение - !snawball Fadiko
Ответ в команде - {username} кинул снежок в {parametr} 
Ответ в чат - <ник пользователя> кинул снежок в Fadiko
